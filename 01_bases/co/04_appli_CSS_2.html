Définition : Le sélecteur

Le sélecteur permet de cibler une balise HTML, de dire quel élément HTML doit être mis en forme.

La syntaxe des sélecteurs peut devenir très complexe.

Définition : Propriété

Taille de police, couleur d'arrière plan, épaisseur de la bordure droite, ...

voici des exemples de propriétés

Définition : Valeur

12 pixels pour la taille de police, arrière-plan rouge, bordure épaisse de 2px, ...

Les propriétés se voient affectées des valeurs, qui peuvent être

une valeur numérique suivie d'une unité (12px, 8em, 50%)

une couleur (#FF0000, rgb(255, 0, 0, 0), red)

une valeur particulière ( capitalize, none, left, right, ...) adaptée à la propriété

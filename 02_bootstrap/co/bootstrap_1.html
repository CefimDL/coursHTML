Qu'est ce que bootstrap ?

Définition : Wikipedia

Twitter Bootstrap est une collection d'outils utile à la création de sites web et applications web. C'est un ensemble qui contient des codes HTML et CSS, des formulaires, boutons, outils de navigation et autres éléments interactifs, ainsi que des extensions JavaScript en option.

Dans les faits, mettre en place un site web avec bootstrap s'avère facile. Il suffit de disposer des bons fichiers, tout se passe ensuite par l'utilisation des classes css appropriées directement dans le code HTML.

Pour ...

Bootstrap est un environnement très utilisé, notamment dans le thème par défaut de prestashop.

Liste de sites réalisés avec bootstrap :

http://expo.getbootstrap.com/

Il permet de facilement créer un site web qui aura une apparence professionnelle.

Par ailleurs il est possible de télécharger des thème alternatifs, qui permettent de personnaliser cette apparence.

... et contre

Le principal défaut de bootstrap, est que le code HTML que l'on doit taper peut vite se retrouver très verbeux.

Ensuite, Framework veut dire "Cadre de Travail". C'est à dire que l'on travail dans un cadre, et que si on veut un résultat très original, il faut en sortir.
